## Getting Started

1) Clone the repo
```sh
git clone https://github.com/Anmewzaa/BasicAPI-101.git
```
2) เปิด terminal เเล้วใช้คำสั่งแล้วติดตั้ง dependencies
```sh
npm i
```
3) เริ่ม project
```sh
npm run dev
```
